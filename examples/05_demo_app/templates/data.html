How it works

Tables are generated using lofigui's table() function with optional headers and styling:

import lofigui as lf

data = [
    ["Alice", "Developer", "5 years"],
    ["Bob", "Designer", "3 years"],
]
lf.table(data, header=["Name", "Role", "Experience"])
