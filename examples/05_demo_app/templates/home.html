Welcome to Lofigui Demo

A comprehensive demonstration of Lofigui features

This demo application showcases the key features of Lofigui, a lightweight Python/Go framework for building simple web GUIs with template inheritance and reactive updates.

What is Lofigui?

Lofigui provides a simple way to create web interfaces with:

  • Template inheritance using Jinja2
  • Automatic page refresh during long-running processes
  • Controller pattern for managing application state
  • Support for Python and Go backends
  • WASM support for running in the browser

Features Demonstrated

Template Inheritance

All pages extend a common base.html template, demonstrating Jinja2 template inheritance with custom blocks.

Data Tables

Display tabular data with automatic formatting using lofigui's table helpers.

Charts & Visualizations

Render charts and visualizations using markdown and HTML output.

Process Management

Run long-running processes with automatic page refresh and status updates.
