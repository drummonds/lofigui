About Lofigui

A lightweight framework for simple web GUIs

Lofigui is a Python/Go framework designed to make it easy to create simple web interfaces with minimal boilerplate.

Key Features

  • Template Inheritance - Build consistent UIs with Jinja2 templates
  • Controller Pattern - Manage long-running processes with automatic refresh
  • Multi-language - Available in both Python and Go
  • WASM Support - Run Go code in the browser via WebAssembly
  • Lightweight - Minimal dependencies, fast startup
  • FastAPI Integration - Works seamlessly with FastAPI

Use Cases

Internal Tools

Build quick admin interfaces and dashboards for internal use

Data Processing

Monitor long-running data processing tasks with live updates

Demos & Prototypes

Quickly prototype web interfaces for proof-of-concepts
