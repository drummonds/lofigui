How It Works

  1. Pyodide Loads: The Pyodide runtime (Python compiled to WASM) loads in a Web Worker
  2. Python Code Loads: Your Python model code is fetched and loaded into Pyodide
  3. Execute on Click: When you click a button, Python code executes in the worker
  4. Results Display: The HTML output is sent back to the main thread and displayed

Benefits

  • No server required - runs entirely in the browser
  • Can be hosted on GitHub Pages for free
  • Python code executes in a Web Worker (non-blocking)
  • Same Python code can run server-side or client-side
