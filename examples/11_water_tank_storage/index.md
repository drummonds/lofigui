# 11 — Water Tank Storage

WASM frontend with a Go API server and persistent state storage. The water tank simulation runs in the browser while data is persisted server-side.

**[Interactivity level](../research-philosophy.html#the-interactivity-spectrum):** 7 — WASM + API server
**[State scope](../research-philosophy.html#the-state-dimension):** Global (the API server holds shared state — all browsers see the same persisted tank). Adding auth would shift this toward Personalised.
