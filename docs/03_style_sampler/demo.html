Technical: WASM + Go templates

This demo runs entirely in the browser. Go's html/template engine is compiled into the WASM binary alongside the templates (embedded via go:embed). Clicking a style button calls goRenderPage() which renders the full page HTML using Go templates — the same code path as the server-side version. Template inheritance ({{block}}/{{define}}) works identically.

No server round-trip. No JavaScript framework. Same Go templates, same rendering engine, running client-side in WebAssembly.
